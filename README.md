#  Recipe Book App

A modern Flutter application that allows users to explore delicious recipes with detailed ingredients, preparation steps, cooking time, and ratings. The app features a clean and responsive user interface designed to provide an enjoyable recipe browsing experience.

## Features

* Browse a collection of recipes
  
* View recipe images and descriptions
  
* Check cooking time and ratings
  
* View detailed ingredients list
  
* Follow step-by-step cooking instructions
  
* Modern and user-friendly UI
  
* Responsive design for different screen sizes

## Screens

### Home Screen

* Displays all available recipes
  
* Recipe cards with image, name, description, cooking time, and rating

### Recipe Detail Screen

* Full recipe image
  
* Detailed description
  
* Ingredients list
  
* Step-by-step cooking instructions
  
* Cooking time and rating information

##  Getting Started

### Prerequisites

* Flutter SDK installed
  
* Android Studio / VS Code
  
* Android Emulator or Physical Device

### Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd recipe-book-app
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
